# Platform_GAME

Управление персонажем с помощью клавиш (стрелочки)
Соберите ВСЕ золотые монеты на уровне
После сбора всех монет автоматически начнется следующий уровень
На каждом уровне больше монет и сложнее препятствия
Избегайте красных препятствий, чтобы не потерять здоровье
Синие щиты дают временную защиту от препятствий

Для запуска игры нажмите кнопку "Начать игру"
